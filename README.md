# Recipe Nutrition Generator & Analyzer

一个集「智能食谱生成」与「专业营养成分分析」于一体的工具，支持自定义食材/口味生成食谱，并自动计算热量、蛋白质、维生素等营养指标。

## ✨ 核心功能
- 📝 基于食材/口味/饮食偏好（如低脂/素食）生成个性化食谱
- 📊 自动分析食谱的营养成分（热量、宏量营养素、微量元素等）
- 📱 支持导出营养分析报告（JSON/CSV格式）
- ⚡ 轻量易部署，支持本地离线运行

## 📋 环境配置
### 前置依赖
- Python 3.10.16
- 推荐使用虚拟环境（conda）

### 安装步骤
# 1. 创建并激活虚拟环境（可选但推荐）
# conda方式
```
conda create -n recipe-nutri python=3.10.16
conda activate recipe-nutri
```
# 2. 安装依赖包
```
pip install -r requirements.txt
```
